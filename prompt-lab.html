프롬프트 실험실

EXAONE의 자유로운 답변 생성 능력을 확인하는 공간입니다. 프롬프트를 입력하고 API 키를 추가하면 바로 응답을 확인할 수 있습니다.

설정 & 프롬프트 입력

API 키가 없으면 데모 응답이 표시됩니다.

404 오류가 나오면 FriendliAI 콘솔의 API Base URL을 확인해주세요.

Failed to fetch가 나오면 네트워크/CORS 차단 여부를 확인하세요.

요청/응답 정보를 확인하려면 체크하세요.

API 키 저장은 브라우저 로컬에만 보관됩니다.
