EXAONE 체험 구성

EXAONE의 성능을 빠르게 확인할 수 있도록 실험 목적별 페이지를 구성했습니다. 각 페이지에서 API 키를 입력하면 실제 FriendliAI API 호출로 결과를 확인할 수 있고, 키가 없다면 로컬 데모 응답으로 흐름을 테스트할 수 있습니다.
